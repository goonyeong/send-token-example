# Blockchain Wallet

- Typescript
- Next js
- React js
- styled components
- Mobx

### 1) Clone

> git clone https://Asher_WM@bitbucket.org/ethan_wemade/blockchain-wallet.git

### 2) Install modules

> npm install

### 3) Build

> npm run build

### 4) Run server with build files (port 3000)

> npm start
